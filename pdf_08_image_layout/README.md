# pdf_08_image_layout

These are standalone programs accompanying the article [Rust: PDFs — Cairo and Pango — Image Block Layout](https://behainguyen.wordpress.com/2026/03/15/rust-pdfs-cairo-and-pango-image-block-layout/).

💡 **Please note**: the code uses hardcoded relative paths, which are not suitable for production. This is for illustration purposes only.

🪟 On Windows, run [set_env.bat](https://github.com/behai-nguyen/polyglot_pdf/blob/main/pdf_08_image_layout/set_env.bat) once before executing `cargo run`.
