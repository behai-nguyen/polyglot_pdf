# pdf_07_image_intro

These are standalone programs accompanying the article [Rust: PDFs — Cairo and Pango — An Introduction to Image Rendering](https://behainguyen.wordpress.com/2026/03/06/rust-pdfs-cairo-and-pango-an-introduction-to-image-rendering/).

💡 **Please note**: the code uses hardcoded relative paths, which are not suitable for production. This is for illustration purposes only.

🪟 On Windows, run [set_env.bat](https://github.com/behai-nguyen/polyglot_pdf/blob/main/pdf_07_image_intro/set_env.bat) once before executing `cargo run`.
