# pdf_09_image_block

These are standalone programs accompanying the article [Rust: PDFs — Pango and Cairo Layout — Supporting Image Blocks](https://behainguyen.wordpress.com/2026/03/25/rust-pdfs-pango-and-cairo-layout-supporting-image-blocks/).

💡 **Please note**: the code uses hardcoded relative paths, which are not suitable for production. This is for illustration purposes only.

🪟 On Windows, run [set_env.bat](https://github.com/behai-nguyen/polyglot_pdf/blob/main/pdf_09_image_block/set_env.bat) once before executing `cargo run`.
